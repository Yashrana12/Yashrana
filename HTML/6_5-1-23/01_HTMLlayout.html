HTML elements are the building blocks of HTML pages.with HTML constructs, images and other objects such as interactive forms may be embedded into the rendered page. HTML provides a means to create structured documents by denoting structure semantics for text such as headings, paragraph, lists, links, quotes and other items. HTML elements are delineated by tag, written using angle brackets.tags such as and directly introduce content into the page.
