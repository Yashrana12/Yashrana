in javascript there is two type of conditional statement

(1) if statement 
    (a) if.. else statement
    (b) if.. else if.. statement
    (c) nested if

(2) switch statement 

syntax :
    (1) if statement
        if(condition){
            statements;
        }
    
    (2) if.. else statement :
        if(condition){
            statements;
        }else{
            statements;
        }

    (3) if..else if.. else statement :
        if(condition){
            statements;
        }else{
            statements
        }    
            else{
                statements
            }

    (4) nested if statement :
        if(condition){
            statements;
        }        else{
            statements;
        }else{
            if(condition){
                statements;
            }else{
                statements;
            }
        }

    (5) switch statement :
        switch(var_name){
            case 1 :
                statement;
                break;
            case 2 :
                statement;
                break;
            .............
            case n :
                statements;
                break;
            default :
                statements; 
                break;            
        }
